# 專題:日系穿搭領航員
## 提案發想
在台灣,日系品牌服飾一直影響台灣人穿衣風格,若能提供消費者日系服飾穿搭建議，推薦消費者目前日本最新時尚,也不用煩惱每天的穿搭

## 目標使用族群
20~35歲
對於日系穿搭有興趣的族群
